# Sony Karakol

Готовый одностраничный сайт и Telegram-бот для игрового клуба Sony Karakol.
Техническое задание взято из прикрепленного PDF: landing page, разделы о клубе,
играх, ценах, галерее и контактах, адаптивность, анимации, lightbox и форма
заявки.

## Структура

- `index.html` - основная HTML-страница.
- `css/styles.css` - оформление, адаптивность и анимации.
- `js/script.js` - меню, активная навигация, счетчики, lightbox и валидация формы.
- `assets/` - локальные изображения сайта.
- `bot/` - Telegram-бот для игр, цен, контактов и бронирования.

## Как открыть сайт

Откройте файл `index.html` в браузере. Сайт не требует сервера и работает как
статическая страница.

## Как запустить Telegram-бота

1. Перейдите в папку `bot`.
2. Выполните `npm install`.
3. Скопируйте `.env.example` в `.env`.
4. Вставьте токен от BotFather в `BOT_TOKEN`.
5. Укажите `ADMIN_CHAT_ID`, если нужно получать заявки администратору.
6. Запустите `npm start`.

Для работы переменных окружения нужен пакет `dotenv`:

```bash
npm install dotenv node-telegram-bot-api
```

После запуска бот покажет меню: игры, цены, контакты, турниры и бронирование.
